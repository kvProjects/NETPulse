NETPulse

A simple Python-based network scanner that identifies active devices on a local network using ARP requests.

## Features
- Scans local network for active devices
- Displays IP and MAC addresses
- Uses ARP packet scanning
- Built with Python and Scapy

## Technologies
Python
Scapy
Networking (ARP protocol)

## Installation

pip install -r requirements.txt

## Usage

python scanner.py

## Example Output

IP Address        MAC Address
-----------------------------------------
192.168.1.1       8c:3b:ad:xx:xx:xx
192.168.1.5       a4:bb:6d:xx:xx:xx

## Learning Purpose

This project demonstrates basic network discovery techniques used in cybersecurity and network administration.
